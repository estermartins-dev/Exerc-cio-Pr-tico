# Exercício 5 - Matrizes em Python

Este projeto consiste em uma aplicação simples em Python para manipulação de matrizes. O objetivo é calcular automaticamente os totais a partir de uma matriz de estoque e uma matriz de preços, utilizando a biblioteca NumPy.

## Funcionalidades

* Cálculo de totais por loja através da multiplicação de matrizes
* Uso da biblioteca NumPy para operações matemáticas
* Exibição do total por loja e total geral

## Estrutura do Projeto

* AULA5_EXERCICIO/: diretório principal
* matrizes.py: código principal
* venv/: ambiente virtual (ignorado no Git)
* .gitignore: arquivos ignorados
* README.md: documentação

## Instruções de Execução

1. Ative o ambiente virtual:
   venv\Scripts\activate

2. Execute o programa:
   python matrizes.py

## Tecnologias Utilizadas

* Python
* NumPy (biblioteca para matrizes)
* Git e GitHub
